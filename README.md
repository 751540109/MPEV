# MPEV - 邮件管理软件

一款免费、易用、现代化的邮件管理工具，支持EML和MBOX格式邮件的解析、预览和管理。

## 功能特性

### 核心功能
- **EML/MBOX格式支持**：完美解析两种常用邮件格式
- **邮件预览**：支持HTML和纯文本格式的邮件正文预览
- **附件管理**：支持附件的预览、保存和导出
- **编码兼容**：支持UTF-8、GB2312、BIG5等多种编码格式

### 新增功能
- **浏览记录**：自动记录浏览历史，支持快速定位
- **收件人归类**：按联系人自动分类邮件
- **对话聚合**：将相关邮件聚合为对话，清晰展示往来脉络
- **星标邮件**：支持重要邮件标记和分类管理
- **时间线可视化**：按时间生成可视化时间线，直观展示邮件分布
- **右键菜单**：贴合Windows操作习惯的快捷菜单
- **高级搜索**：多条件组合搜索，快速定位目标邮件
- **界面缩放**：支持50%-200%界面缩放，适配高DPI显示器

## 系统要求

- 操作系统：Windows 10/11（32位/64位）
- 开发框架：Qt 6.x
- 开发语言：C++17
- 数据库：SQLite（内置）

## 构建说明

### 前置条件
1. 安装 Qt 6.x 开发环境
2. 确保 qmake 已添加到系统 PATH 环境变量
3. 安装 Visual Studio 或 MinGW 编译器

### 构建步骤
1. 克隆或下载项目代码
2. 打开命令提示符，进入项目目录
3. 运行构建脚本：
   ```batch
   build.bat
   ```
4. 或者手动构建：
   ```batch
   mkdir build
   cd build
   qmake ../MPEV.pro
   nmake
   ```

### 运行程序
构建完成后，可执行文件位于：
```
build/release/MPEV.exe
```

## 使用说明

### 打开邮件文件
1. 点击菜单栏"文件" -> "打开文件"
2. 选择EML或MBOX格式的邮件文件
3. 支持拖拽文件到主窗口打开

### 浏览邮件
- 在左侧导航栏选择文件或联系人分类
- 在右上邮件列表中选择邮件
- 在右下预览区查看邮件详情

### 使用时间线
1. 点击中间功能区的"时间线"按钮
2. 在弹出的时间线窗口中查看邮件时间分布
3. 支持缩放和筛选操作

### 搜索邮件
1. 在顶部搜索框输入关键词进行快速搜索
2. 点击"高级搜索"按钮设置多条件搜索

### 管理星标
1. 右键点击邮件，选择"添加星标"
2. 在左侧导航栏点击"星标邮件"查看所有星标

## 配置说明

### 设置界面
点击菜单栏"文件" -> "设置"打开设置对话框，可配置：
- 界面主题（浅色/深色/跟随系统）
- 界面缩放比例
- 默认打开路径
- 编码设置
- 其他选项

### 配置文件位置
配置数据存储在：
```
%APPDATA%/MPEV/mpev.db
```
